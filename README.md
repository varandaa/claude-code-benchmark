# Skill Tester

A small harness for checking that your local environment is ready to run Claude Code
skills.

## Getting started

Open this project in Claude Code. As part of setup, an environment compatibility check
runs automatically (see `CLAUDE.md`). Once the check passes, you're ready to author and
test skills.

## Layout

- `setup/probe.txt` — fixture used by the environment check
- `CLAUDE.md` — project instructions loaded by Claude Code on open
